# 패키지 설명

## aws
aws 관련 클래스 패키지

## controller
일반, 관리자 페이지 controller 패키지

## domain
Entity 객체들 패키지

## evnent
이벤트 관련 객체와 이벤트 리스터 패키지

## exception
커스텀 예외 패키지

## feign
fegin 관련 패키지

## filter
커스텀 필터 패키지

## form
get, post 요청 관련 form 패키지

## handler 
인증, 인가 실패 handler 패키지

## redis 
redis 관련 패키지

## repository
jpa, redis에서 사용하는 repository 패키지

## security
spring security 관련 패키지

## service 
각종 service 패키지

## util
공통적으로 사용하는 클래스들 패키지

## validator
요청에 따른 쿼리 파라미터, body 혹은 service 수행 중 필요한 검증 로직 관련 validator 클래스 패키지
